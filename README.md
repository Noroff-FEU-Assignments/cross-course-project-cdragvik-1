# RainyDays

## Description 
This cross course project has been part of the first school year in a variety of courses. Is is an online shop selling mens and womens rain jackets. They are designed for a range of different outdoor activities that enriches people's lives. The target audience is men and women aged 30 to 50 whose interests are: being outdoors, hiking, exploring, skiing, camping, canoeing. The jackets are mid-range in price and emphasize durability and being suitable for a range of different weather types ensuring customers are comfortable whatever adventure they go on.

The building of the site started with HTML and CSS only, before adding to it with JS. 

The site architectur is as following: 
- Home 
- List of jackets 
- Jacket specific page 
- Checkout page
- Checkout success
- About 
- Contact 


## Built with 
HTML, CSS and JS.
